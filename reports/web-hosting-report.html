What is a web server and why would you need one?

A web server can refer to two entities, either on the hardware side, the software side or both. On the hardware side a web server is a computer that holds the web server software and the actual physical web page components. The hardware is a computer specifically built to be a web server and host webpages. A common misconception is that web server computers are high powered super computers. These servers are connected to the internet and handle physical data exchange with other physical devices. The software is also a simple application that sends the web-page to the web-client requesting it. Servers use HTTP’s that understands URL’s and HTTP. Content is accessed through the requested domain name and then sent to the requesting device being used to browse the content.

Explain why getting a domain name is not enough to put your site up on the web.

While a domain name gives your site it’s unique name, web servers give you a place to store the physical files and be accessible to the internet from a computer that is always available to the internet. Basically, the domain is the name, the web server is the storage and dispatch system.

What service does a web hosting company provide?

Web hosting companies provide the web servers which store the websites. They also provide the physical environment that the web servers require, including a cooled environment and repairs and housekeeping as required. They also provide the technical knowhow to do these things to reduce downtime of web sites. ”A web hosting company provides the web server and internet connectivity that allows your website to be available to anyone connected to the internet.” freeservers.com web hosting provider

What are the most important characteristics to look for in selecting a good web hosting company?

According to johnnyflash.com the top issue is reliability, simply having the site stay online and not go down. The article also lists speed and bandwidth, how fast can the visitors get to the site and basically how wide is the portal through which the data travels to them. The smaller the bandwidth, the slower things become. Picture traffic piling up on the highway. A two-lane highway with a lot of fast cars is going to be backed up like Seattle traffic during…any moment in time, vs an 8-lane highway with lots of space for all those cars. The wider the band with the more and larger files can go through it. Also listed is support, because when you need help, it should be there and pricing that isn’t going to be changed or added to constantly. In other words, standard pricing.

What are the most important features for you when reviewing a hosting plan?

I feel that there seems to be a couple very important features that are needed in a hosting plan. First would be speed and space and security. Not running out of room for my files and having my site visitors avoid experiencing a slow webpage are important. And having a site become cancerous due to malware or getting hacked is just plain terrible.

Just as important would be uptime. If the server goes down and the site goes with it, then that’s time for the site to either not be helpful to those seeking information or a loss of consumers and protentional customers. Which might be the main reason for the entire website.

Also support. If there is an issue or a problem or a question, the site hosting company needs to be available.

Finally, pricing. While it would be lovely to get all of this for free, you get what you pay for. Realizing that, I would want to see all this and other features (tech, free apps etc.) and find them at the fairest pricing per month.
